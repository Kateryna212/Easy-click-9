# 🛒 Easy-click Marketplace — Оптимізація та Доступність

Вебзастосунок (SPA) навчального маркетплейсу, повністю модернізований та оптимізований в межах практичних робіт №8 та №9. 

## ⚡ Результати аудиту в Google Lighthouse (Chrome/Edge DevTools)
Завдяки впровадженим покращенням, застосунок досяг найвищих результатів швидкості та інклюзивності:
* **Performance (Продуктивність):** `100 / 100` 🚀
* **Accessibility (Доступність):** `100 / 100` ♿
* **Best Practices (Кращі практики):** `100 / 100` 🏆

---

## 🔧 Що було реалізовано в Практичній роботі №8 (Оптимізація швидкості):
1. **Мінімізація DOM-операцій:** Застосовано патерн `DocumentFragment` для збирання карток товарів у пам'яті комп'ютера, що зменшило кількість рендерингів сторінки (Reflow) з 4-х до 1-го комплексного запиту.
2. **Оптимізація зображень:** Реалізовано перехід на сучасний стиснутий формат зображень товарів (.webp).
3. **Ледаче завантаження (Lazy Loading):** Інтегровано атрибут `loading="lazy"` для завантаження контенту тільки при скролі.
4. **Усунення зсувів макета (CLS = 0):** Фіксовані розміри `width` та `height` для картинок заздалегідь резервують під них місце на екрані.

---

## ♿ Що було реалізовано в Практичній роботі №9 (Доступність / Accessibility):
1. **ARIA-семантика:** Додано системні ролі (`role="banner"`, `role="main"`, `role="contentinfo"`) та детальні мітки `aria-label` для кнопок покупки, щоб Screen Readers чітко озвучували контекст.
2. **Якісні описи `alt`:** Кожне зображення отримало унікальний альтернативний опис замість загальних назв.
3. **Контрастність кольорів (WCAG AAA):** Кольори шрифтів, кнопок та фону підібрано відповідно до стандартів інклюзивності та чіткого розрізнення кольорів.
4. **Повноцінна навігація з клавіатури:** Реалізовано можливість повного керування маркетплейсом через клавішу `Tab` з яскравим візуальним фокусом (`:focus`).
